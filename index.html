WWE2K CAW Name Splicer Companion App

About

A simple companion app for the create-a-wrestler for WWE2K series that allows users to randomly generate names for their custom created wrestlers (CAWs).

Languages: HTML, CSS & JavaScript

Instructions

Step Description
1 Click the "Choose File" button to upload a ".txt" or ".csv" file with names from the CAW mode in your WWE2K game.
The names should be seperated by a new line with only one name on each line
2 Once you select your file and upload it to the app, a table of unique names is
dynamically created from your name list with no repititions.
3 Scroll through your newly generated name list and check any names in the far right
"Delete" column that you would want to exclude from your list.
4 Once you make it all the way down to your name list, click the "Delete Selected Names"
button and click the "Download As PDF" button.
5 The file automatically downloads the .pdf file to your default downloads folder on your browser
You are all set!
6 To start over, simply refresh the page in the browser



Have fun and enjoy!!!
